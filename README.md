📊 Análise de Dados – E-commerce Brasileiro (Olist)

Este projeto tem como objetivo realizar uma análise completa de dados de e-commerce brasileiro, utilizando o conjunto de dados público da Olist, amplamente utilizado em estudos e portfólios de Data Analytics.

O foco do projeto é demonstrar habilidades práticas em Python (Pandas), SQL Server e Power BI, cobrindo todas as etapas do processo analítico: ingestão, tratamento, modelagem, análise e visualização de dados.

🎯 Objetivos do Projeto

Realizar o tratamento e limpeza de dados reais de e-commerce

Construir métricas e KPIs de negócio

Criar um modelo analítico utilizando SQL

Desenvolver um dashboard interativo no Power BI

Demonstrar um fluxo de trabalho completo (end-to-end) em análise de dados

🗂️ Dataset

Fonte: Brazilian E-Commerce Public Dataset by Olist

Plataforma: Kaggle

Link: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Os dados representam aproximadamente 100 mil pedidos realizados entre 2016 e 2018, contendo informações sobre pedidos, produtos, clientes, pagamentos, avaliações e logística.

⚠️ Devido ao tamanho dos arquivos, os dados brutos e tratados não estão incluídos neste repositório.
